# Enterprise CI/CD Pipeline for Spring Boot using GitLab CI/CD
Overview

This project demonstrates an end-to-end CI/CD pipeline for a Spring Boot application using GitLab CI/CD. The pipeline automates source code testing, security scanning, application packaging, Docker image creation, image publishing, and automated deployment using Docker Compose on a self-hosted Linux runner.

The objective of this project is to demonstrate practical DevOps concepts and automate the complete software delivery lifecycle.
